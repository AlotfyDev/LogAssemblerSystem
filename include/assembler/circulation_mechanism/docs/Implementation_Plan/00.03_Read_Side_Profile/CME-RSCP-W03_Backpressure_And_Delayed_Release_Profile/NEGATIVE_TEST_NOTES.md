# CME-RSCP-W03 Negative / Anti-Collapse Notes

1. Backpressure profile is not a throttling algorithm.
2. Delayed release profile does not release material.
3. Post-bridge capacity profile does not allocate containers.
4. Persistence completion trigger does not implement persistence.
5. Receiver acknowledgement trigger does not call receiver.
6. Read-side pressure must not be pushed back into pre-bridge CME by default.
