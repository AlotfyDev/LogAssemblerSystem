# CME-W01 Negative / Anti-Collapse Notes

This wave is too foundational for runtime negative tests, but it protects:

1. IDs are not runtime actors.
2. Registry index is not a container reference by itself.
3. Slot state is distinct from container state.
4. Container state is distinct from round state.
5. Diagnostic severity/category are not logging implementation.
6. Foundational states do not execute transitions.
