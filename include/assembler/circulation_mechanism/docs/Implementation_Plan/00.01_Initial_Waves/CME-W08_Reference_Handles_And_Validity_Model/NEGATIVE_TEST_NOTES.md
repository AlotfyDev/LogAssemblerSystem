# CME-W08 Negative / Anti-Collapse Notes

This wave protects the following rules:

1. Ingress references cannot be consumed as dispatch references.
2. Dispatch references cannot be consumed as ingress references.
3. Access ref existence does not imply validity.
4. Reference identity is not container ownership.
5. Next-ref token is not a locator.
6. Stale reference report is not recovery behavior.
7. Dispatch reference is not a bridge protocol handle.
