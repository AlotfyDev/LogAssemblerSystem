# CME-W10 Negative / Anti-Collapse Notes

This wave protects the following rules:

1. Ingress is not Log API.
2. Ingress does not assemble envelopes.
3. Ingress does not calculate references.
4. Ingress access policy does not mutate slots.
5. Boundary descriptor does not expose container/slot internals.
6. Placement report is not dispatch report.
7. Ingress does not own bridge/read-side lifecycle.
