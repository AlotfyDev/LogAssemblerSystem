## Scope

This wave catalogs:

- optional add-on families for the policy subsystem,
- add-on ownership boundaries,
- candidate future waves,
- non-collapse rules.

It does not implement any add-on, validator, runner, exporter, realizer, persistence engine, or runtime component.
