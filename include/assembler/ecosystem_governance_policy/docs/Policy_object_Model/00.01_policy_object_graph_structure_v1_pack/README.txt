# Policy Object Graph Structure V1 Pack

## Included files
1. `01_Policy_Object_Graph_Structure_V1.md` — the Markdown draft
2. `02_policy_object_graph_structure_v1.dot` — Graphviz source
3. `02_policy_object_graph_structure_v1.png` / `.svg` — rendered Graphviz image
4. `03_policy_object_graph_structure_v1.mmd` — Mermaid source
5. `03_policy_object_graph_structure_v1.png` / `.svg` — rendered Mermaid image (or fallback image if Mermaid renderer was unavailable)
6. `04_policy_object_graph_structure_v1_figma_ready.json` — Figma-ready structured spec
7. `04_policy_object_graph_structure_v1_figma_ready.svg` / `.png` — generated visual based on the Figma-ready spec

## Notes
- A native `.fig` Figma file is not generated in this environment.
- Instead, a Figma-ready structured JSON spec is provided, plus a rendered SVG/PNG visual that matches it.
- The graph in this pack is structural only. Traversal/orchestration semantics are intentionally deferred.
